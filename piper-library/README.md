# Piper Jenkins 2.0 pipeline library

Convenience Step Library for Jenkins 2.0 Pipelines with Pipeline Templates

## For details and documentation follow this [Link](https://url.sap/piper)

If you have a question or if you need assistance please address it to the [respective channel](https://github.wdf.sap.corp/ContinuousDelivery/piper-library/issues/new/choose).

## Getting Involved

**We are happy to see contributions!**
This could be:

* providing feedback, bugs, suggestions, ... using the issue functionality within GitHub
* further improvement of the pipeline templates
* extending the currently tackled use cases

Please see the [contribution guidelines](https://github.wdf.sap.corp/ContinuousDelivery/piper-library/blob/master/.github/CONTRIBUTING.md) for further information.
