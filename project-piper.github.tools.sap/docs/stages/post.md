## ${docGenStageContent}

### sapCumulusUpload

The following files are generated inside the `sapPiperPipelineStagePost` step and uploaded to Cumulus, if configured:

- `cumulus-configuration.json` from jenkins workspace

- `env.json` containing environment variables `BUILD_URL`, `GIT_BRANCH`, `GIT_COMMIT`, `GIT_URL`

- `lock-run.json` to automatically [lock](https://wiki.wdf.sap.corp/wiki/x/GUtWiw) a pipeline run in Cumulus when the artifacts are promoted

- `jenkins-log.txt` generated by `jenkinsMaterializeLog`

(Compare [Cumulus Integration](./cumulus-integration.md))

## ${docGenStageActivation}

The stage is enabled automatically inside the general purpose pipeline.

## ${docGenStepActivation}

The steps referenced in the stage content description are activated automatically.

## ${docGenStageParameters}

The stage does not take any configuration parameters. For configuration of the steps executed please refer to the respective step documentation.

## ${docGenStageConfiguration}

The post stage can not be customized. The cumulus upload of files will be enabled by configuring cumulus properly.
