# How to enable OPA5

Same as [QUnit](testing-qunit.md), use Karma.

<!-- markdownlint-disable-next-line MD036 -->
*SAP UI5 Support Assistant*

is a code quality checker delivered by the UI5 team within the UI5 framework. Besides running it from within a UI5 application it can also be automated and executed within the OPA5 test framework as shown in the [demo kit example](https://sapui5.hana.ondemand.com/sdk/#/entity/sap.ui.test.Opa5/sample/sap.ui.core.sample.OpaWithSupportAssistant/code/Opa.js).
An internal test implementation can be found [here](https://github.wdf.sap.corp/D065687/Pipeline-Test-MTA/blob/D054628-OPA-support-assistant/shoppinglist-ui/webapp/test/integration/ListReportTest.js).
