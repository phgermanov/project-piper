## SAP-Specifics

If source/target registry is SAP artifactory then targetRegistryUrl should be as described [here](https://pages.github.tools.sap/Common-Repository/Artifactory-Internet-Facing/how_to/how_to_repo_url/:)

- `<Repository_Name>.common.repositories.cloud.sap`

For example: [targetRegistryUrl: https://piper-test.common.repositories.cloud.sap] <br>

[Introducing imagePushToRegistry step news article](https://pages.github.tools.sap/project-piper/news/2024/03/22/imagepushtoregistry-to-strengthen-the-gitops-deployment-via-piper/)
