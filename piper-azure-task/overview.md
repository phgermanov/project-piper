# Piper Azure DevOps Task

This extension provides a wrapper task for the [*Project Piper* CICD steps](https://www.project-piper.io).

## How to use

After installing the extension, you can add the task to [your pipeline](https://docs.microsoft.com/en-us/azure/devops/pipelines/?WT.mc_id=DOP-MVP-5001511&view=azure-devops).

## Available tasks

Azure DevOps

* **Piper**: executes a Piper step

## Get the source

The [source](https://github.tools.sap/project-piper/piper-azure-task) for this extension is on GitHub. Take, fork, and extend.
